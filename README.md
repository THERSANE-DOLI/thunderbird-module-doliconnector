# How to build package xpi
## On linux 
run command or execute as a program
```bash
buildXpi.sh
```
## On other Os
### 1. Build a Zip of the Folder's files
```
cd my-addon-folder
zip -r my-addon.zip *
```
### 2. Rename to `.xpi`
```
mv my-addon.zip my-addon.xpi
```
